# CHO细胞培养优化系统

> 基于机器学习和模型预测控制的CHO细胞培养补料策略优化系统

## ✨ 系统特点

- 🎯 **完整工作流**：数据上传 → 模型配置 → 训练 → 结果分析 → 补料策略
- 🎨 **美观界面**：Vue3 + Element Plus，专业渐变色主题
- 🚀 **快速部署**：一键启动，自动安装依赖
- 📊 **实时可视化**：ECharts交互式图表
- 🔧 **灵活配置**：支持多种模型类型和参数调整
- 💾 **结果导出**：一键下载补料策略CSV

## 🏗️ 系统架构

```
┌─────────────────┐      ┌─────────────────┐      ┌─────────────────┐

```


## 🚀 快速开始

### 1. 一键启动（推荐）

双击运行 `start.bat`，系统会自动启动并打开浏览器。

### 2. 构建前端（生产）

```bash
cd frontend
# 本机部署可直接构建
npm run build

# 如需让前端固定请求某个服务器地址，可在构建前注入：
# Windows PowerShell:
$env:VITE_API_BASE_URL="http://服务器IP:8000"
npm run build
```

### 3. 手动启动（开发）

**后端：**
```bash
python app.py
```

**前端：**
```bash
cd frontend
npm install  # 首次运行
npm run dev
```

访问：http://localhost:5173

开发模式下，前端默认通过代理访问 `http://localhost:8000`。如需改为其它后端地址，可在启动前设置环境变量：

```powershell
$env:VITE_DEV_API_PROXY_TARGET="http://服务器IP:8000"
.\start_frontend.ps1
```


## 📖 使用流程

### 步骤1：上传数据
- 支持CSV格式
- 必须包含：`run_id`, `t_h`
- 建议包含：VCD, Glc, Gln, Amm, Lac, Titer, temp, pH

### 步骤2：配置模型
- **软传感器**：预测VCD和Titer
- **动态模型**：预测下一时刻状态
- **MPC参数**：预测时域、控制时域、权重

### 步骤3：训练模型
- 自动执行完整训练流程
- 实时显示进度和日志
- 约1-3分钟完成

### 步骤4：分析结果
- 查看模型性能（R²、RMSE等）
- 可视化状态和控制轨迹
- 交互式图表

### 步骤5：获取策略
- 查看详细补料建议
- 下载CSV格式补料计划
- 操作指南
